# Multi-Store Support

## Description
Allow users to be members of multiple stores:
- Single user account can have memberships in multiple stores
- Switch between stores easily
- Unified points view across all stores (optional)
- Store-specific offers and products

## Implementation
- Already supports multiple memberships via user_store_memberships
- Add store switcher in app header
- Unified balance view option
- Cross-store rewards (spend at partner stores)

## UI Elements
- Store selector dropdown
- "All Stores" aggregate view
- Store-specific points history

## Priority
High
